# XAIOBS
xampp all in one bash script
---
### start XAMPP
``$ sudo ./xaiobs start``
### stop XAMPP
``$ sudo ./xaiobs stop``

### restart XAMPP
``$ sudo ./xaiobs restart``

### create localhost
``$ sudo ./xaiobs mklh <name>``

### Configure
to configure other params you may change variables inside the script

in future other features to controll xampp will be added
